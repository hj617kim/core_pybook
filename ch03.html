3. 기초 I(기본 자료형, 변수, 함수)¶

3.1. 기본 자료형¶

자료형Data Type은 컴파일러나 인터프리터에게 데이터를 어떻게 사용할 것인지 알려주는 데이터 속성 중 하나다.
파이썬에서 지원하는 몇 가지 주요 자료형을 간단히 살펴보자.

  • 숫자형 Numeric Types

    • 정수형 Integral

      • 정수 int

      • 불리언 bool

    • 부동소수점 float

  • 문자열 str

  • None

3.1.1. 정수¶

일반적으로 알고 있는 정수int(자연수, 0, 음의 정수)들의 자료형이다. 정수형으로 나타내면 덧셈, 뺄셈, 곱셈, 나눗셈 등의 기본 연산이 가능하다.
예를 들어, …, -3, -2, -1, 0, 1, 2, 3, …은 모두 정수형(int)이다.

2 + 3 - 4

1

4 * 2

8

3.1.2. 부동소수점¶

부동소수점float은 원래 실수를 컴퓨터에서 다루기 위해 개발되었으나 실제로는 유리수 일부만을 다룬다. 무리수인 원주율 \(\pi\)의 경우에도 컴퓨터의 한계로 인해 소수점 이하 적당한 자리에서 끊어서 사용한다. 예를 들어, 1.2, 0.333, -1.2, -3.7680, 4.0, .7 등은 모두 부동소수점형(float)이다.

1.2 + 3.5

4.7

참고

정수부나 소수부가 0이라면 생략하고 작성할 수 있다. 예를 들어, 0.7은 .7로, 1.0은 1.이라고 작성할 수 있다.

>>> 0.7 == .7
True
>>> 1.0 == 1.
True

지수 표현 방식

e나 E 다음에 오는 수는 10의 지수가 된다. 예를 들어, 30000.0은 3e4 또는 3E4라고 작성할 수 있다.

>>> 30000.0 == 3e4
True
>>> 30000.0 == 3E4
True

이렇게 표현된 수는 부동소수점이다.

부동소수점의 한계

부동소수점은 십진 소수를 정확하게 표현하지 못한다. 예를 들어, 0.1 + 0.1 + 0.1을 계산하면 0.30000000000000004으로 0.3이 같은지를 확인하면 False가 나온다.

>>> 0.1 + 0.1 + 0.1
0.30000000000000004
>>> 0.1 + 0.1 + 0.1 == 0.3
False

3.1.3. 문자열¶

문자열str은 문자들을 나열한 값들을 일컫는 자료형으로 작은 따옴표(') 또는 큰 따옴표(")를 사용한다.

'Hello, World'

'Hello, World'

"Hello, World"

'Hello, World'

Example 3.1

Hello, "World"를 보이는 그대로 출력하는 코드를 작성하여라.

문자열 내부에서 큰 따옴표를 사용하고 싶을 때는 작은 따옴표로 문자열을 감싸면 된다.

print('Hello, "World"')

Hello, "World"

반대로 문자열 내부에서 작은 따옴표를 사용하고 싶을 때는 큰 따옴표로 문자열을 감싸면 된다.

print("I'm a student")

I'm a student

백슬래시(\, 한글 키보드에서는 원화기호￦를 사용)를 사용하여 문자열 내부에서 사용되는 따옴표의 특수 역할을 해제escape할 수 있다.

print("Hello, \"World\"")

Hello, "World"

여러 줄로 이루어진 문자열은 삼중 따옴표(''' ''' 또는 """ """)를 사용할 수 있다.

print('''
Hello,
 World
''')

Hello,
 World

print("""
Hello,
 World
""")

Hello,
 World

3.1.3.1. 문자열에 특수 문자 활용하기¶

백슬래시(￦), 줄바꾸기(￦n), 탭(￦t) 등은 문자열에 사용될 경우 특수한 기능을 갖는다.

print("Hello\n World")

Hello
 World

print("Hello\t World")

Hello	 World

백슬래시(\)기호를 붙여서 특수 기능을 해제escape할 수 있다.

print("Hello\\n World")

Hello\n World

print("Hello\\t World")

Hello\t World

print("Good\\night")

Good\night

연속된 백슬래시 두 개를 출력하려면 아래와 같이 해야 한다.

print("\\\\")

\\

주의

아래와 같이 코드를 작성하면 오류가 발생하므로 주의해야 한다.

>>> print("\\\")
  File "/tmp/ipykernel_76/2337434698.py", line 1
    print("\\\")
                ^
SyntaxError: EOL while scanning string literal

순수 문자열

‘가공되지 않은’의 의미를 갖는 ‘raw’ 단어의 첫 글자인 ‘r’을 문자열 앞에 두면 특수 기능이 사라진다.

print(r"Hello\n World")

Hello\n World

print(r"Hello\t World")

Hello\t World

print(r"Hello\ World")

Hello\ World

3.1.3.3. f-string¶

문자열 앞에 f를 붙이면, 문자열 포매팅 string formatting을 사용할 수 있다.

문자열 포매팅 string formatting

문자열 포매팅이란 문자열 안에 값을 삽입하는 방법이다.

예제와 함께 살펴보자.

name = '강현'
f'{name}님, 안녕하세요.'

'강현님, 안녕하세요.'

age = 3
f'강현이는 {age}살이다.'

'강현이는 3살이다.'

name = '강현'
age = 3
f'{name}이는 {age}살이다.'

'강현이는 3살이다.'

{}안에 변수와 수식(+, -, *, / 등)을 함께 사용하는 것도 가능하다.

name = '강현'
age = 3
f'{name}이의 동생은 {age - 2}살이다'

'강현이의 동생은 1살이다'

소수점 표현

콜론(:) 뒤에 소수점아래 몇 번째 자리까지 출력할지를 적어주면, 그 만큼을 보여준다.
예제와 함께 살펴보자.

  • .은 소수점을 의미하고, 소수점 뒤의 숫자는 소수점 뒤에 나올 숫자의 개수다.

num = 0.123456789
print(f'{num:.1f}')
print(f'{num:.2f}')
print(f'{num:.3f}')
print(f'{num:.5f}')  # 소수점 아래 여섯 번째 자리에서 반올림

0.1
0.12
0.123
0.12346

문자열 정렬

num1 = 0.12
num2 = 0.12345
num3 = 0.12345678

print(f"He paid {num1} dollars.")
print(f"He paid {num2} dollars.")
print(f"He paid {num3} dollars.")

He paid 0.12 dollars.
He paid 0.12345 dollars.
He paid 0.12345678 dollars.

  • 콜론: 뒤에 <은 왼쪽 정렬, >은 오른쪽 정렬, ^은 가운데 정렬을 의미한다.

  • 정렬 문자(<, >, ^) 뒤에 숫자만큼의 공간을 사용한다.

  • 정렬 문자(<, >, ^) 앞에 문자를 적으면 그 문자로 공백을 채운다.

print(f"He paid {num1:>10} dollars.")
print(f"He paid {num2:>10} dollars.")
print(f"He paid {num3:>10} dollars.")

He paid       0.12 dollars.
He paid    0.12345 dollars.
He paid 0.12345678 dollars.

print(f"He paid {num1:<10} dollars.")
print(f"He paid {num2:<10} dollars.")
print(f"He paid {num3:<10} dollars.")

He paid 0.12       dollars.
He paid 0.12345    dollars.
He paid 0.12345678 dollars.

print(f"He paid {num1:^10} dollars.")
print(f"He paid {num2:^10} dollars.")
print(f"He paid {num3:^10} dollars.")

He paid    0.12    dollars.
He paid  0.12345   dollars.
He paid 0.12345678 dollars.

print(f"He paid {num1:.^20} dollars.")
print(f"He paid {num2:.^20} dollars.")
print(f"He paid {num3:.^20} dollars.")

He paid ........0.12........ dollars.
He paid ......0.12345....... dollars.
He paid .....0.12345678..... dollars.

소수점 표현과 정렬을 동시에 사용할 수도 있다.

print(f"He paid {num:*>10.2f} dollars.")
print(f"He paid {num:*>10.3f} dollars.")
print(f"He paid {num:*>10.4f} dollars.")

He paid ******0.12 dollars.
He paid *****0.123 dollars.
He paid ****0.1235 dollars.

3.1.4. 불리언¶

불리언bool자료형은 참(True)과 거짓(False)를 나타내는 자료형이다. 파이썬에서 불리언 자료형은 int 형의 자식형 subtype이고, 대부분의 상황에서 True와 False는 각각 1과 0처럼 동작한다.

3.2. 변수¶

3.2.1. 변수와 변수할당¶

변수variable는 (컴퓨터 메모리에) 저장된 하나의 값을 가리키는 이름을 말한다.

3.2.1.1. 변수명¶

변수명은 임의로 정할 수 있지만 할당되는 값과 연관된 이름을 사용할 것을 권장한다.
또한 아래 제한 사항을 지켜야 한다.

  • 반드시 영어 알파벳 문자(a-z, A-Z) 또는 밑줄기호(_)로 시작해야 하며, 이후에는 알파벳, 숫자(0-9), 밑줄기호가 임의로 사용될 수 있다.

  • 파이썬 키워드keyword를 변수 이름으로 사용하면 안된다.

  • 대소문자를 구분해야 한다: YOU, you, You, yOu는 모두 다른 이름으로 처리된다.

  • -, +, *, / 등의 연산자 기호는 이름에 사용할 수 없다.

  • @, $, ? 등의 기호도 사용되지 않는다.

  • 변수명에 공백을 사용할 수 없다.

3.2.1.2. 변수 할당 variable assignment ¶

변수를 선언하려면 해당 변수가 하나의 값을 가리키도록 하는 변수 할당을 명령해야 한다.

주의

변수가 정의되지 않고(값을 할당하지 않고) 사용하려고 하면 오류가 발생한다.

변수 선언과 할당은 동시에 이루어지며, 변수명이 등호기호(=) 왼편에, 변수와 연관된 값value을 표현하는 표현식expression은 등호 기호 오른편에 위치한다.

변수 = 값

num = 5
pi = 3.1415926535897932
name = '강현'

  • num 변수가 가리키는 값은 정수 5이다.

  • pi 변수가 가리키는 값은 부동소수점 3.1415926535897932이다.

  • name 변수가 가리키는 값은 문자열 '강현'이다.

선언된 변수는 할당된 값과 동일하게 취급된다.

'안녕, ' + name

'안녕, 강현'

2 * num

10

변수가 가리키는 값은 변경될 수 있다. 예를 들어, 변수 num에 17을 새롭게 할당하면, num 변수가 가리키는 값은 정수 5가 아니라 정수 17이다.

print(num)
num = 17
print(num)

5
17

세미콜론

모든 명령문은 한 줄에 완성하는 것이 기본이며, 연속된 명령문은 줄바꿈을 해서 작성한다. 하지만 매우 간단한 명령문은 한 줄에 연속으로 작성할 수 있는데, 이때 세미콜론(;)을 사용한다.

a = 1; b = 2; c = 3

print(a)
print(b)
print(c)

1
2
3

세미콜론을 사용하기보다는 각각의 명령문을 줄바꿈하여 명확하게 작성하는 것을 추천한다.

다중 할당 multiple assignment

다중할당은 동시에 2개 이상의 값을 2개 이상의 변수에 할당하는 것을 말한다.

a, b, c = 2, 4, 6

print(a)
print(b)
print(c)

2
4
6

변수 스왑 variable swap

일반적으로 두 변수를 스왑할 때 임시변수 temporarily variable를 사용한다.

a = 1
b = 2
print('a :', a, ', b :', b)

temp = a
a = b
b = temp

print('a :', a, ', b :', b)

a : 1 , b : 2
a : 2 , b : 1

하지만 파이썬은 임시변수를 사용하지 않고 아래와 같이 스왑할 수 있다.

a = 1
b = 2
print('a :', a, ', b :', b)

a, b = b, a
print('a :', a, ', b :', b)

a : 1 , b : 2
a : 2 , b : 1

3.2.2. 동적 타이핑과 정적 타이핑¶

3.2.2.1. 동적 타이핑과 정적 타이핑¶

파이썬은 동적 타이핑 Dynamic typing을 지원한다. 동적 타이핑은 변수를 미리 선언하지 않고, 변수를 생성하고자 할 때 값을 초기화하여 사용하는 것을 말한다. 변수의 자료형은 파이썬이 알아서 판단하기 때문에 동적 타이핑은 프로그래밍을 처음 접하는 사람들에게 편리하다.

동적 타이핑과는 달리 변수를 사용하기 전 미리 선언해야 하는 정적 타이핑 Static Typing도 있다. C나 자바처럼 주로 컴파일 방식의 언어가 정적 타이핑을 지원한다. 이처럼 사용할 변수와 변수에 할당될 값의 자료형을 미리 선언하는 것은 불편해 보일 수도 있다. 하지만 정적 타이핑은 컴파일하는 과정에서 선언한 자료형에 맞는 값을 할당하였는지, 선언 후 사용하지 않는 변수가 있는지를 확인하여 버그 발생 확률을 낮출 수 있다는 장점이 있다. 이에 동적 타이핑을 지원하는 파이썬에서도 타입 힌트 Type Hints를 명시하기도 한다.

3.2.2.2. 타입 힌트¶

파이썬 3.5이후 버전에서는 타입 힌트 또는 타입 어노테이션 Type Annotation을 사용할 수 있다.

다음과 같이 변수명 뒤에 콜론(:)을 붙이고 타입을 적어주는 방식으로 타입 힌트를 추가한다.
예를 들어, a : int, b : float, c : bool, s : str등과 같이 사용한다.

num : int = 5
pi : float = 3.1415926535897932
name : str = '강현'

print(num)
print(pi)
print(name)

5
3.141592653589793
강현

주의

타입 힌트의 사용은 실제 코드 실행에 아무런 영향을 미치지 않는다.

예를 들어, 변수 a에 타입 힌트와는 다른 형type의 값을 할당하여도 오류가 발생하지 않는다.

a : int = 3.4
print(a)
print(type(a))

3.4
<class 'float'>

mypy

타입 검사기를 사용하면 코드가 실행될 때 타입 힌트에 오류가 있는지를 확인할 수 있다. 현재 많이 사용되고 있는 파이썬 타입 검사기 중 하나는 mypy마이파이이다.

주피터 노트북 jupyter notebook 에서는 아래의 코드를 사용하여 mypy를 설치하여 사용할 수 있다.

>>> pip install mypy-ipython  
>>> %load_ext mypy_ipython

그런 다음 아래와 코드를 작성하고 타입 검사기를 사용해보자.

>>> a : int = 3.4
>>> print(a)
3.4
>>> print(type(a))
<class 'float'>

mypy를 사용하면 타입 힌트가 작성된 파이썬 파일의 코드 전체에 타입 힌트 오류가 있는지를 확인할 수 있다.

>>> %mypy
    a : int = 3.4
error: Incompatible types in assignment (expression has type "float", variable has type "int")
Found 1 error in 1 file (checked 1 source file)
Type checking failed

여기서는 타입 힌트와 일치하지 않는 값이 할당되었다는 메시지와 함께 오류가 발생하였다.

3.3. 함수¶

프로그램은 명령문의 합성으로 구현된다. 그런데 특정 기능을 수행하는 명령문의 일부가 너무 길거나, 반복적으로 사용되어 보다 편하게 사용하고 싶거나 프로그램을 보다 체계적으로 구현하도록 하고 싶을 때가 있다. 그럴 때는 특정 기능을 수행하는 명령문에 이름을 주고 필요에 따라 해당 이름을 대신 활용할 수 있다.

함수는 함수의 이름에 가려진 명령문을 대행하며 해당 명령문을 실행하려면 지정된 함수를 호출 call 하면 된다. 함수 호출 function call 은 함수 이름과 연관된 명령문을 실행하도록 하는 일종의 명령문이다. 함수 호출의 일반적인 형식은 다음과 같다.

함수이름(인자1, 인자2, ..., 인자n)

3.3.4. print() 함수¶

그 동안 출력을 위해 print() 함수를 사용하였다. 예를 들어, Hello, python을 출력하고 싶다면, 아래와 같이 코드를 작성하면 된다.

print('Hello, python')

Hello, python

이제 print() 함수에 대해서 조금 더 살펴보자.

  • 큰 따옴표(또는 작은 따옴표)로 둘러싸인 문자열을 연속해서 사용하면 문자열에서 +연산을 한 것과 동일한 결과를 출력해준다.

print("Hello ""World")
print('Hello ''World')
print('Hello '"World")
print("Hello " + "World")

Hello World
Hello World
Hello World
Hello World

  • 콤마(,)로 구분하여 여러 개의 인자를 사용할 수 있다.

print("Hello", "World")

Hello

 World

print("강아지", 3, "마리")

강아지 3 마리

  • 인자들 사이에 구분자를 넣고 싶다면, sep을 변경하면 된다. sep의 기본값은 공백이다.

print("010", "1234", "5678", sep = "-")
print("Hello", "World", sep = ", ")

010-1234-5678
Hello, World

  • 마지막에 출력할 문자를 변경하고 싶다면, end를 변경하면 된다. end의 기본값은 줄변경(\n)이다.

print("a")
print("b", end = "")
print("c", end = "\t")
print("d")

a
bc	d

참고

print() 함수에 의해 화면에 출력되는 문자열은 반환값이 아니고 화면에 지정된 값을 출력하는 명령문의 실행 결과에 불과하다. 함수의 반환값은 변수에 할당해서 사용할 수 있지만 화면에 출력된 문자열은 그럴 수 없다는 데서 print() 함수는 특정한 반환값을 생성하지 않고, 대신에 None이 반환값으로 사용된다는 것을 확인할 수 있다.

x = print(123) # 아래 123은 변수 x가 가리키는 값이 아니다. 

123

print(x) #변수 x는 None을 가리킨다.

None
